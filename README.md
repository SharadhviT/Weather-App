#Weather App 🌤️
A simple, responsive web-based Weather Application that lets you check the current weather of any city or your current location. Built using HTML, CSS, and JavaScript, and powered by the OpenWeatherMap API.
Features
Search for weather by city name.
Use your current location to get local weather.
Displays temperature, humidity, wind speed, and weather description.
Shows weather icons for quick visual reference.
Search history: Quickly click past searched cities (last 5).
Responsive design for desktop and mobile devices.

Getting Started
1. Clone the Repository
git clone https://github.com/YOUR_USERNAME/weather-app.git
cd weather-app
2. Get OpenWeatherMap API Key
Sign up at OpenWeatherMap.
Get your free API key.
3. Add API Key
Open script.js
Replace YOUR_API_KEY_HERE with your API key:
const apiKey = "YOUR_API_KEY_HERE";
4. Run Locally
Open index.html in your browser.
Search for any city or click Use My Location.
